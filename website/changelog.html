Historial de Versiones

Registro transparente de todos los cambios de MeaCore

v1.2.2 - Alfa Release 🚀 (Actual)

Novedades:

  • CI/CD Unificado: Flujo de trabajo automatizado para generar instaladores de Linux y Windows sincronizados con la versión del proyecto.
  • Documentación Profesional: Reescritura total del README y listado de características técnicas.

v1.2.1 - GNOME & Assets Fix 🐧

  • Dock de GNOME: Parche manual del archivo .desktop en el proceso de CI/CD para garantizar que el icono se agrupe correctamente en el dock de Ubuntu.
  • Filtro de Assets: Optimización del algoritmo de descarga para conservar únicamente archivos de lenguaje en Español e Inglés, reduciendo el peso innecesario.

v1.2.0 - Modloaders & Java Integration 🛠️

  • Soporte Multi-Java: Detección inteligente de versiones de Java necesarias para diferentes versiones de Minecraft.
  • Motores de Mods: Soporte inicial para la inyección de Forge y Fabric.
  • NBT Sync: Sincronización automática de listas de servidores multijugador directamente en los archivos del juego.

v1.1.8-alpha - Gestión de Almacenamiento 🛡️

  • Eliminación Permanente: Nueva funcionalidad para borrar perfiles y sus datos residuales (screenshots, saves, logs) de forma segura.

v1.1.7-alpha - Telemetría y Alertas 📡

  • Advertencias de RAM: Implementación de alertas en tiempo real cuando la memoria libre es insuficiente para el perfil seleccionado.
  • Identidad de Dock: Refuerzo de la identidad visual en Ubuntu 24.04.

v1.1.0 - El Nacimiento de MeaCore ✨

  • Rebranding: Transición oficial a la marca MeaCore Launcher.
  • Multi-Instancia: Aislamiento total de directorios de juego por perfil.
  • Diagnóstico de Sistema: Integración con OSHI para telemetría avanzada de hardware y optimización automática de JVM (ZGC/Aikar's Flags).
